# Stock Portfolio Tracker (Python)

A simple console-based stock portfolio tracker developed using Python as part of the CodeAlpha Python Programming Internship (Task 2).

## Features
- Predefined stock prices
- Add stock and quantity
- Calculate total investment value
- Save portfolio as TXT or CSV file

## Concepts Used
- Dictionary
- Input/output
- Basic arithmetic
- File handling

## How to Run
python stock_portfolio_tracker.py
